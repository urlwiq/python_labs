**Цели:** Освоить базовые механизмы ввода/вывода данных, методы форматирования строк.

**В ходе выполнения лабораторной работы были успешно закреплены на практике следующие навыки:**
- Работа с функциями input() для ввода данных и print() для их вывода.
- Преобразование типов данных (int(), float()).
- Обработка пользовательского ввода (замена запятой на точку для корректного преобразования в число).
- Использование f-строк для эффективного и наглядного форматирования вывода.
